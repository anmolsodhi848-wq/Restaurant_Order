# **Restaurant_Order**


## **Project Objective**

The objective of this project is to analyze order-level sales data using Excel and Tableau to uncover actionable insights that improve business decision-making and sales performance.

## **Process**

1. Collected and reviewed restaurant order transaction data.
2. Cleaned data in Excel by removing duplicates, correcting data types, and standardizing item names and categories.
3. Imported the cleaned dataset into Tableau for analysis.
4. Built calculated measures using SUM, COUNT, and COUNTD.
5. Designed interactive dashboards using bar charts, line charts, pie charts, box plots, and scatter plots.
6. Enabled category- and item-level comparisons to analyze trends and performance.

## **Project Insights**

1. Italian and Asian cuisines generate the highest revenue among all categories.
2. Asian cuisine records the highest number of orders, indicating frequent customer preference.
3. Items like Korean Beef Bowl, Spaghetti & Meatballs, and Tofu Pad Thai are top revenue contributors.
4. Items such as Edamame and Hamburger are the most frequently ordered.
5. Sales remain relatively consistent over time with noticeable peak ordering hours.
6. A positive correlation exists between item popularity and revenue generation.


## **Conclusions**

1. The restaurant maintains a balanced sales mix across multiple cuisines.
2. High-frequency, moderately priced items significantly drive overall revenue.
3. Time-based sales trends provide clarity on demand patterns.
4. Data visualization using Tableau effectively converts raw data into actionable insights.
